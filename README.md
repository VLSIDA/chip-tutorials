# UCSC VLSI-DA Chip Tutorials

Open-source chip-design tutorials covering SPICE, OpenROAD / ORFS,
LibreLane, STA, Sky130, GF180MCU, and KLayout.

**Read the tutorials here:** <https://vlsida.github.io/chip-tutorials/>

To clone this repo locally:

```bash
git clone https://github.com/VLSIDA/chip-tutorials.git
cd chip-tutorials
git pull     # if already cloned, keep up to date
```
